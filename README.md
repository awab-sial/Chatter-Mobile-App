## 📌 Overview

**Chatter** is a sleek, real-time messaging application developed using **Flutter** and **Firebase**, designed as a semester project for the **Mobile App Development** course at **Bahria University**. The app supports one-on-one chat with advanced media functionalities including image sharing, emoji reactions, and voice messaging.

From authentication to media delivery, every feature is optimized for responsiveness and usability, making **Chatter** a compact yet powerful modern messaging solution.

---

## ✨ Features

* 🔐 **Firebase Authentication** (Email signup/login)
* 💬 **Real-Time One-to-One Chat** using Firestore
* 📸 **Image Sharing** with upload preview
* 🎤 **Voice Messages** (record, send, and playback)
* 💗 **Double-Tap Animated Heart** for chat reactions
* 😊 **Emoji Support** for expressive conversations
* 🖼️ **Profile Picture Upload** and update
* 🌐 **Data Persistence** using Firebase Firestore & Storage
* 📱 **Beautiful Responsive UI** built with Flutter

---

## 🧱 Project Goals

* Create a production-level messaging app in Flutter
* Learn and integrate Firebase services (Auth, Firestore, Storage)
* Implement voice and image media support
* Build responsive UI/UX with animations
* Enable real-time chat sync using Firestore streams

---

## 🔍 Project Scope

* One-to-one chat system with secure authentication
* Image and audio media sharing functionality
* Local voice recording and streaming playback
* Seamless UI flow between splash, login, signup, and chat screens
* Persistent session storage using `shared_preferences`
* Interactive animations for better user engagement

---

## 🛠️ Project Requirements

* Flutter SDK (>= 3.0.0)
* Firebase Auth, Firestore, Storage
* `image_picker`, `firebase_core`, `firebase_auth`, `cloud_firestore`, `firebase_storage`
* `flutter_sound` or `just_audio` for voice messaging
* `shared_preferences` for local session management
* Device permissions for microphone, gallery access
